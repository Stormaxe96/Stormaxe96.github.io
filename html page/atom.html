Ce este Atom?










    Cosmos, cunoscut și sub numele de Atom, este un proiect de blockchain open-source care își propune să ofere o soluție de interoperabilitate între diverse rețele blockchain. A fost fondat în 2014 de Jae Kwon și Ethan Buchman și a fost lansat oficial în martie 2019.
    Una dintre caracteristicile cheie ale proiectului Cosmos este arhitectura sa de tip hub-and-spoke, care permite conectarea mai multor rețele blockchain, numite zone, la un hub central numit Cosmos Hub. Această arhitectură permite transferul de valori și date între diverse rețele blockchain fără a fi nevoie de o interfață centralizată.
    Proiectul Cosmos utilizează un algoritm de consens numit Tendermint, care utilizează un set de validatori pentru a valida tranzacțiile și a administra rețeaua. De asemenea, Cosmos oferă un set de instrumente pentru dezvoltatorii de aplicații blockchain, inclusiv un kit de dezvoltare software (SDK) și un limbaj de programare numit Solidity++, care permite dezvoltatorilor să creeze aplicații inteligente (smart contracts) pe rețeaua Cosmos.
    Token-ul nativ al rețelei Cosmos este numit Atom și este utilizat pentru a plăti taxele de tranzacționare, pentru a accesa serviciile de pe rețea și pentru a participa la procesul de validare a tranzacțiilor. Atom este disponibil pentru tranzacționare pe mai multe platforme de schimb de criptomonede și are o capitalizare de piață semnificativă.
